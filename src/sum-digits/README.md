# Sum of Digits

## Description

Write a function that takes a number and returns the sum of all its digits. If the input is negative, use the absolute value.

## Examples

- Input: 123
- Output: `6` (1 + 2 + 3)

- Input: 9119
- Output: `20` (9 + 1 + 1 + 9)

- Input: -456
- Output: `15` (4 + 5 + 6)

- Input: 0
- Output: `0`

## Learning Goals

- Practice number manipulation
- Learn to work with digit extraction
- Use modulo and division operators
- Handle edge cases (negative numbers, zero)

## Function Signature

```typescript
export function sumOfDigits(num: number): number
```

## Testing

Create your solution in a student folder (e.g., `student1/sum-of-digits.ts`) and write tests in `student1/sum-of-digits.test.ts`.
