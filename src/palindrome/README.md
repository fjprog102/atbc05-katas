# Palindrome Checker

## Description

Write a function that determines whether a given string is a palindrome. A palindrome reads the same forwards and backwards, ignoring spaces, punctuation, and letter casing.

## Examples

- Input: "A man, a plan, a canal: Panama"
- Output: `true`

- Input: "hello"
- Output: `false`

- Input: "racecar"
- Output: `true`

## Learning Goals

- Practice string manipulation methods (toLowerCase, replace, etc.)
- Work with character iteration
- Understand two-pointer technique
- Handle edge cases (empty strings, special characters)

## Function Signature

```typescript
export function isPalindrome(str: string): boolean
```

## Testing

Create your solution in a student folder (e.g., `student1/palindrome.ts`) and write tests in `student1/palindrome.test.ts`.
